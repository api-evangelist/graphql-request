# GraphQL Request Plans

GraphQL Request (now Graffle) is a free, open-source MIT-licensed library. There are no paid tiers or commercial subscription plans for the library itself. All features are available to everyone at no cost.

## Open Source (Free)

- Full access to all library features
- HTTP and in-memory transport support
- TypeScript type inference and Document Builder
- Extension system (OpenTelemetry, file uploads, schema error handling)
- Custom scalar codec support
- Spec-compliant GraphQL over HTTP
- GraphQL Multipart Request specification support
- Community support via GitHub Issues

**Installation:** `npm install graffle@next graphql`
**npm (legacy):** `npm install graphql-request graphql`
